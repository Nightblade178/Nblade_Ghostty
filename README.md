# NB_Ghostty
This is my own personalised Ghostty Config. I want to keep the keybinds intuitive and closer to vim if possible. I do not like the default multi switch keybindings. I think 1 combo should be used as activator like Ctrl+Shift as Ctrl+B is to Tmux so i have customized it to be that.

Use Ghostty-Launch to deactivate the GTK's  input-method framework. It conflicts with Ctrl+Shift+u.
